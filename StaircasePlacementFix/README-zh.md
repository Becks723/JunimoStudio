# 楼梯修复
修复矿井部分地皮无法放置楼梯的问题。


## 下载 & 安装

1. 下载并安装[SMAPI](https://smapi.io/)。
2. [下载](https://www.nexusmods.com/stardewvalley/mods/13381)本模组。解压并放入Mods文件夹。
* _(Android)_ 开头的文件为安卓版本。
